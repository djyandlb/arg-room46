### 解密方式

#### 摩斯码 → 搜索【回音】

```
[页面] surface/echo.html

[推理] ① 这段摩斯码不是正常的帖子内容——是故意留下的
       ② 解密大师已经解码出"WATCHING"——但我可以亲自验证
       ③ rules.html的控制台也暗示了摩斯码的存在
[行动] ① 在搜索框尝试与帖子主题相关的词
       ② 帖子主题是"听到回音"→ 搜索 "【回音】"
[产出] → 跳转 decode/decode-morse.html（见第2阶段）
```

#### Base64 → 搜索【V4cXy】

```
[页面] surface/browser.html（查看页面源码 Ctrl+U）
[看到] HTML注释中有一段奇怪的文字：
       <!-- VjQ3IGxlZnQgeW91IGEgY2x1ZS4gU2VhcmNoICJWNENYeS4i -->
[推理] ① 普通注释不会由大写字母和数字混合组成
       ② 末尾的"=="是Base64编码的典型特征
       ③ 需要用Base64解码工具解码
[行动] ① 在浏览器控制台输入：atob("VjQ3IGxlZnQgeW91IGEgY2x1ZS4gU2VhcmNoICJWNENYeS4i")
       ② 解码结果："V47 left you a clue. Search **'V4cXy'**."
       ③ 在搜索框输入 "【V4cXy】"
[产出] → 跳转 decode/decode-base64.html（见第2阶段）
```

#### 反转文字 → 搜索【永远不会醒来】或【那扇门】

