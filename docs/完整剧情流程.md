## 一、解密总链一览

```
论坛首页 → 浏览→发现异常帖 → 解码→得到关键词 → 搜索→新页面 → 更多线索 → 组合密钥 → 镜像论坛 → 里论坛 → 真相

每个页面内容配比：
  ▪ 70% 真实论坛内容（灌水、讨论）
  ▪ 20% 隐藏线索（源码注释、控制台、格式异常）
  ▪ 10% 可提取的关键词/密钥片段
```

## 二、世界线设定

**2004年6月** — "回声茶馆"论坛上线。同月，服务器无意被 ARPANET 时代遗留的实验协议"LAMP46"感染。该协议寄生在 DNS 解析层。

**2004-2017年** — LAMP46 在论坛数据库中伪装成隐藏版块。论坛上开始出现"看到陌生页面"的帖子。

**2017年8月14日** — 用户 V47 发现了真相。他在技术角落发帖揭露。46 小时后账号被"回收"。帖子被删，但数据库缓存保留了一份。

**2026年** — 论坛早已荒废，转成只读归档。但 LAMP46 仍然在线等你。

## 四、详细解密流程

### PHASE 1 — 进入论坛

#### 📄 forum.html — 论坛首页

**Y2K 视觉：** phpBB 风格。深蓝渐变顶栏、白色版块列表。

**正常内容：**
版块：闲聊区(46,046帖)、技术角落(12,401帖)、生活笔记(8,234帖)、归档区(446帖)、站务管理(46帖)
页脚：`总帖子数 46,046 · 总会员数 4,604 · 在线 46 人（隐身 4 人）`

**隐藏：**
- HTML 注释：`<!-- 归档区的帖子好像少了几页 -->`
- 控制台：`◈ 归档区比看起来的多东西。试试管理员权限？`
- 在线人数永远是 46 的倍数

**所有帖子行都带 onclick 跳转。** 有的链到灌水帖，有的链到线索帖。

### PHASE 3 — 教程帖

#### 📄 newbie.html ⭐

"萌新小透明"的新人帖。老玩家"解密大师"在第 4 楼回帖介绍四种加密：
> ① 摩斯密码：`... --- ...` 就是 SOS。
> ② Base64：`SGVsbG8=` 就是 Hello。
> ③ 凯撒密码：echo 移 3 位变成 hfkr。
> ④ 倒序文字："你好吗"→"吗好你"。

**必读帖。不读此页无法解后面的谜。**

可通过搜索"教程""密码""新人"随机匹配到此页。

## 七、灌水帖 vs 线索帖

只有带 ⭐ 的页面产出关键词。灌水帖不推进主线，但填充世界感。
玩家在不带 ⭐ 的帖子找线索是正常体验——"在千篇一律里找到异常"本身就是 ARG 的乐趣。
