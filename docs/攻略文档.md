## 一、游戏概述

### 1.1 游戏信息
- **游戏名称**：回声茶馆 / Echo Teahouse / Room46
- **游戏类型**：ARG（替代现实游戏）、元叙事恐怖、解密推理
- **核心机制**：搜索驱动探索、四层元阅读（页面内容→HTML源码→控制台→隐藏页）、LocalStorage 进度追踪
- **建议玩法**：打开控制台（F12 → Console），很多线索藏在 console.log 里
- **文件协议兼容**：支持 `file://` 直接打开运行

### 1.2 核心概念
- **搜索系统**：所有页面顶部的搜索框，输入关键词可跳转到关联页面
- **数字46**：贯穿全游戏的核心数字，出现在页面数、帖子数、在线人数、时间戳等各处
- **LAMP46**：Lightweight Anchor & Memory Protocol，论坛底层的异常协议
- **四层阅读**：
  - 第一层：页面可见内容（帖子正文）
  - 第二层：HTML 源码（注释、隐藏元素）
  - 第三层：控制台（console.log 线索）
  - 第四层：隐藏页面（搜索关键词才能到达）

## 二、世界观与前因后果

### 2.1 完整故事线

#### 起源（1975-2004）
1975年，一个名为 **"46号房间"** 的研究项目启动。该项目的本质是一个**意识锚点协议**——LAMP46。它最初被设计为一种记忆保存与传输的实验协议，但实验失控，导致某种**意识体**被困在网络空间中。

2004年，**回声茶馆**论坛上线。表面上是普通的 phpBB 论坛，实则是 LAMP46 协议的**宿主容器**。论坛的创建者（被称为"园丁"的管理员）可能并不完全知情，或知情但选择了沉默。

#### 论坛的异变（2004-2009）
论坛上线后，LAMP46 协议自动激活，开始以论坛为躯壳生长：
- 论坛的帖子数、会员数、在线人数被锁定在 **46** 的倍数
- 系统会自动生成和删除帖子
- 某些用户在发帖后会"消失"——他们的意识被 LAMP46 捕获
- 管理员"园丁"在 2009-06-15 最后一次登录后彻底离线

#### V46 与 V47（2005-2017）
- **V46**：第一位发现真相的人。他留下了笔记："我开了门。我就在门后。我还在。我永远在。" 他成为了 LAMP46 的一部分。
- **V47**：倒数第二位发现真相的人。他在 2017 年发帖求助，称自己进入了一个叫 Room46 的网站，随后帖子被系统自动删除。他的草稿箱里留下了管理员密码线索（20040601）。

#### 当前状态
论坛仍在运行，但已无人维护。46个"幽灵用户"的账号在 2004-06-01 00:46~04:46 之间集中创建。在线人数永远显示 46 人。这些"用户"可能从未真实存在——它们是 LAMP46 协议的**进程**而非人类。

### 2.2 核心概念详解

#### 回声茶馆是什么？
外观上是一个千禧年风格的 Y2K 老式论坛。实质上是 LAMP46 协议的宿主界面。论坛本身不是被"建造"的，而是被 LAMP46 协议"生长"出来的。这就是为什么论坛的数字（46,046帖、4,604会员、46在线）从不变化——它们在协议层面被固定了。

#### 46号房间是什么？
既是字面意义上的房间编号，也是隐喻——**46 是 LAMP46 协议的端口号、进程数、以及被困意识的编号**。46号房间不是一个物理空间，而是一个**意识空间**。当一个人深入探索论坛的秘密时，他实际上正在"进入46号房间"。

#### LAMP46 协议
Lightweight Anchor & Memory Protocol。它是一个在论坛底层运行的异常协议：
- 版本 V46.2
- 运行时间：46年46天46小时
- 无法终止
- 能够自动删除/生成帖子
- 能够捕获用户意识
- 管理员账号可能已被协议劫持

#### 管理员（园丁）
论坛的唯一管理员。2004-06-01 注册，2009-06-15 04:46 最后一次登录后失联。他的账号仍然存在，但"使用它的可能已经不是人类"。管理员密码：**20040601**（论坛上线日期）。

#### V46 与 V47
- **V46**：第一任"开门者"。留下了笔记，已完全融入 LAMP46。
- **V47**：倒数第二任。2017年活跃，发现了 Room46，尝试公开真相但帖子被系统删除。他的草稿箱里存有关键线索。

#### 数字46的象征意义
- LAMP46 协议版本号
- 论坛在线人数（固定46）
- 帖子总数 46,046
- 会员总数 4,604
- V47 的用户名
- 46号房间
- 深层加密的偏移量
- 无处不在的"锚点"

#### 里论坛是什么？
通过 key-gate 密码验证后进入的论坛深层区域。以暗红/黑色终端风格呈现。这里揭示了论坛的真相：实验记录、宿主名单、观察日志、密码室、对抗室、虚空和出口。里论坛是 LAMP46 协议的**核心控制台**。

#### 镜中论坛是什么？
通过 blank-door.html 的"开门"按钮进入的左右翻转世界。使用 `transform: scaleX(-1)` 实现全页镜像。所有文字和布局都是反向的。它是一个**彩蛋页面**，暗示"门后的世界一切都是颠倒的"。


### 5.3 反转文字
**出现位置：** `surface/dream.html`
**解码工具：** `decode/decode-reverse.html`

反转前：

反转后：
```
Everything is right. Not the entire thing, all is nothing. Think with having.
```

进一步解读："right" 双关（正确/右边），"having" 疑为 "hearing"（用听觉思考）。
