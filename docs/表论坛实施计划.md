# Room46 V2 · 表论坛 实施计划（修正版）

## 核心机制

```
forum.html（论坛首页）
  │
  ├── 可点击的帖子（intro层，12页）
  │   ├── 灌水帖 × 6（无关键词，纯世界填充）
  │   ├── newbie.html（教程帖，教加密）
  │   ├── echo.html（线索帖#1：藏着摩斯密码）
  │   ├── browser.html（线索帖#2：藏着Base64）
  │   └── dream.html（线索帖#3：藏着倒序文字）
  │
  └── 🔍 搜索功能（核心）
        ├── 搜「回响」→ decode-morse.html（首页看不到）
        ├── 搜「V4cXy」→ decode-base64.html（首页看不到）
        ├── 搜「永远不会醒来」→ decode-reverse.html（首页看不到）
        ├── 搜「残骸」→ decode-base64.html（首页看不到）
        ├── 搜「站务管理」→ admin.html（首页看不到）
        ├── 搜「Outsider」→ decode-caesar.html（首页看不到）
        ├── 搜「anchor_0」→ key-gate.html（首页看不到）
        ├── 无关搜索 → 跳转灌水帖
        │
        └── 搜索历史面板（记录玩家搜过的词）
```

## 文件分层

### A层 - 首页可见（forum.html可点击直达）
| 页码 | 文件 | 类型 | 产出 |
|------|------|------|------|
| 首页 | forum.html | 论坛主页 | 搜索入口 |
| 表01 | rules.html | 灌水 | 无 |
| 表02 | food.html | 灌水 | 无 |
| 表03 | tech.html | 灌水 | 无 |
| 表04 | movie.html | 灌水 | 无 |
| 表05 | travel.html | 灌水 | 无 |
| 表06 | diary.html | 灌水 | 无 |
| 表07 | newbie.html | 教程 | 四种加密知识 |
| 表08 | echo.html | 线索 | 摩斯→关键词"回响" |
| 表09 | browser.html | 线索 | Base64→关键词"V4cXy" |
| 表10 | dream.html | 线索 | 倒序→关键词"永远不会醒来" |

### B层 - 仅搜索可见（论坛首页无链接）
| 页码 | 文件 | 到达方式 | 产出 |
|------|------|---------|------|
| 表11 | decode-morse.html | 搜"回响" | 密钥A + 关键词"残骸" |
| 表12 | decode-base64.html | 搜"V4cXy"或"残骸" | 密钥B + 关键词"站务管理" |
| 表13 | decode-reverse.html | 搜"永远不会醒来" | 密钥名称列表 |
| 表14 | admin.html | 搜"站务管理" | 被删帖入口 |
| 表15 | deleted.html | 从admin进入或搜"被删" | 凯撒→"Outsider" |
| 表16 | decode-caesar.html | 搜"Outsider" | 密钥C + "anchor_0" |
| 表17 | key-gate.html | 搜"anchor_0" | 密钥组合→镜象论坛 |
